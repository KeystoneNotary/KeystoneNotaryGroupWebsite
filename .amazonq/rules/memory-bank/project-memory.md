## Technical Stack & Architecture
- **Project Structure**: Single-page application with dark glassmorphism theme
- **Development Environment**: Windows-based development
- **Version Control**: Git-enabled project
- **AI Integration**: Amazon Q Developer enabled
- **Frontend**: HTML5, CSS3 (Grid, Flexbox, Custom Properties), Vanilla JavaScript
- **Animations**: GSAP 3.12.5 with ScrollTrigger and ScrollToPlugin
- **Design**: Dark glass theme with parallax effects and scroll animations
